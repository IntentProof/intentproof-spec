# Minimal Required Template

Reference ID: `reference.templates.minimal-required.v1`

This template pack demonstrates the smallest complete reference policy pack:
a single required event rule, one happy-path fixture, and one expected run. It
is intended as the copyable shape for future curated packs, not as a customer
vertical policy.

Use this pack when adding a new reference policy family and you need the
required file layout, manifest fields, fixture naming, and validation behavior.
